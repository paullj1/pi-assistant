# Pi Assistant

Wake-word voice assistant with local Whisper, barge-in, and optional MCP tools.

## Run

- `pi-assistant`

## Install

From GitHub:

```bash
curl -fsSL https://raw.githubusercontent.com/paullj1/pi-assistant/refs/heads/main/install.sh | bash
```

## Config

Config lives at `~/.config/pi-assistant/config.toml` by default (XDG compatible).
Environment variables override values in the TOML file.
